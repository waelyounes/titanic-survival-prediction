# Titanic - Machine Learning from Disaster 🚢

### 🏆 Project Overview
This project is my first milestone in Data Science. It aims to predict the survival of passengers on the Titanic using Machine Learning algorithms. 

### 🧬 Professional Background
As a PhD holder in **Mathematics and Logic**, I approached this challenge by focusing on:
- **Logical Feature Engineering:** Creating meaningful features like `FamilySize`.
- **Mathematical Imputation:** Using statistical measures to handle missing values in 'Age'.
- **Model Evaluation:** Analyzing the trade-off between Precision and Recall.

### 🛠️ Tools & Technologies
- **Language:** Python
- **Environment:** Google Colab / Kaggle API
- **Libraries:** Pandas, NumPy, Seaborn, Scikit-learn

### 📊 Methodology
1. **Exploratory Data Analysis (EDA):** Visualizing correlations between gender, class, and survival.
2. **Data Cleaning:** Handling missing values in Age, Embarked, and Cabin columns.
3. **Model Selection:** Comparing **Random Forest** and **Logistic Regression**.
4. **Validation:** Achieved an accuracy that earned me the **Python Coder Badge** on Kaggle.
